Ayuda Para registrar tu asistencia en la clase sigue los siguientes pasos: Alumno

1.- Al abrir la Aplicación deberás seleccionar el tipo de usuario que te corresponda, en este caso debes elegir "Alumno"

2.- Una vez seleccionado el tipo de usuario deberás ingresar con tus credenciales.

Ejemplo:

Usuario: ejemplo@duocuc.cl

Contraseña: contraseña123

3.- Selecciona la opción QR asistencia.

4.- A continuación deberás usar la cámara para escanear el código QR, una vez posicionado el teléfono debes presionar el botón "Escanear código":

5.- Siguiendo estos pasos tu asistencia quedará registrada.
